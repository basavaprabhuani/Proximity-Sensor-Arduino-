# Proximity-Sensor-Arduino-

The proximity sensor is a project I built using Arduino (UNO R4 Minima) 

The system detects nearby objects using ultrasonic waves and provides real-time feedback through a display, LED, and buzzer.

The module uses the following components: 
- HC-SR04 Ultrasonic Pulse-Echo Sensor
- Liquid Crystal Display (LCD)
- Active Buzzer
- Red LED
